# Hackathon Projects

Is repo me multiple chhote HTML/CSS/JavaScript projects hain. Har project ka code alag folder me hai aur browser me seedhe open karke chalaya ja sakta hai.

**Tech Stack**
- HTML
- CSS
- JavaScript

**Projects**

1. Basic Tip Calculator Project
- Path: `Basic Tip Calculator Project /index.html`
- Feature: Bill amount, tip % aur number of people ke base par per-person tip aur total calculate karta hai.
- Files: `index.html`, `style.css`, `script.js`
- https://jovial-pony-177b8b.netlify.app/

1. To-Do List
- Path: `To-Do List/index.html`
- Feature: Task add, delete, aur click par completed (line-through) mark.
- Files: `index.html`, `script.js` (UI JS se dynamically create hota hai)
- https://dapper-medovik-d02846.netlify.app/

1. Blog Post
- Path: `Blog Post/blog.html`
- Feature: Static blog post page with header, article, image, blockquote, author bio, aur comments form.
- https://enchanting-dragon-c63c3c.netlify.app/

1. Event Page
- Path: `Event /Event.html`
- Feature: Event details, schedule, speakers, highlights aur registration link.
- https://fanciful-cobbler-b4e1ae.netlify.app/

1. Feedback Form
- Path: `Feedback/index.html`
- Feature: Name, email, rating aur comment wala feedback form.
- Files: `index.html`, `style.css`
- https://rococo-pie-6cee47.netlify.app/

1. Product Card UI
- Path: `Product Card UI/index.html`
- Feature: E‑commerce style layout, category filters, product cards, cart section, banner.
- Files: `index.html`, `style.css`
- https://eclectic-bubblegum-b48414.netlify.app/#

1. Food Recipe
- Path: `food recepi/index.html`
- Feature: Food items list with images aur prices.
- Extra Page: `food recepi/contect.html` (contact form)
- https://eclectic-bubblegum-b48414.netlify.app/

**Run/Use Kaise Karein**
1. Kisi bhi project ke HTML file ko browser me open karein.
1. JS wale projects (Tip Calculator, To‑Do List) direct browser me kaam karte hain.

**Notes**
- Kuch folder names me space hai (jaise `Basic Tip Calculator Project ` aur `Event `), isliye path copy karte waqt exact name use karein.
